# Savishkaar-CUK
Savishkar | CUK Innovation Hub 🚀
Savishkar is a high-performance landing page designed for the Innovation Hub at the Central University of Karnataka (CUK). It serves as a digital gateway for students and researchers to explore core research domains, upcoming robotics workshops, and leverage AI for academic brainstorming.

✨ Features
AI Research Catalyst: Integrated with Google Gemini 1.5 Flash to generate novel research proposals for B.Tech projects instantly.

Modern UI/UX: Built with a "Glassmorphism" design aesthetic, smooth scroll animations, and a responsive mobile-first layout.

Dynamic Sections:

Core Domains: Highlights Sustainable Energy, AI, Robotics, and IoT.

Event Tracker: Showcases recent and upcoming workshops with status indicators (e.g., Completed, Starting Nov 18th).

Interactive Stats: Highlighting labs, projects, and patent counts.

Tailwind Powered: Rapidly styled using Utility-first CSS for maximum performance and easy maintenance.

🛠️ Tech Stack
Frontend: HTML5, Tailwind CSS (via CDN)

Typography: Google Fonts (Roboto)

AI Integration: Google Generative AI (Gemini API)

Animations: Custom CSS Keyframes & Tailwind Transitions
